YAXHA is a modern web application designed to help users prepare for the IELTS Speaking exam. Inspired by the elegance of the **Sarus Crane**, the application provides a calm, professional, and minimalist environment for users to practice their speaking skills.

It acts as an interactive AI examiner that asks standard IELTS questions, listens to your responses in real-time, and plans to provide feedback based on fluency, vocabulary, grammar, and pronunciation.

## âœ¨ Key Features

- **Interactive AI Agent**: A personable "Crane AI" agent that simulates a real examiner, complete with dynamic visuals.
- **Real-time Audio Visualization**: Features a sophisticated Gemini-style audio visualizer that responds to voice input, providing immediate visual feedback.
- **Stage-based Exams**: Structured exam flow starting with standard introduction questions (Stage 1).
- **Minimalist UI**: A clean, distraction-free interface built with **Tailwind CSS**, focused on user experience and professional aesthetics.
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.

## ğŸš€ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

1. **Clone the repository**
   ```sh
   git clone https://github.com/your-username/YAXHA.git
   cd YAXHA
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Start the development server**
   ```sh
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173` (or the port shown in your terminal).
