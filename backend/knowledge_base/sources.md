# Official IELTS Sources & Legitimate Documentation

This document catalogs the authoritative sources used to build and verify the IELTS Speaking AI Agent knowledge base.

## Authoritative Documentation [OFFICIAL]
1.  **IELTS Guide for Teachers (2021 Update):** Core reference for test structure, assessment criteria, and scoring arithmetic.
    - *Source Type:* Official IDP/British Council Publication.
2.  **IELTS Speaking Band Descriptors (Public Version):** The gold standard for scoring (Fluency, Lexical, Grammar, Pronunciation).
    - *Source Type:* Official IELTS Partnership Document.
3.  **IELTS Examiner Quality Assurance (Professional Standards):** MPRs (Minimum Professional Requirements) for certified examiners.
    - *Source Type:* Cambridge Assessment English Standards.
4.  **IELTS Research Reports Online Series:** Academic validation of test reliability and analytical rating scales.
    - *Source Type:* Official Peer-Reviewed Research (Published by IELTS).

## Legitimate Preparation Materials [PREPARATION]
1.  **IELTS-Speaking-Part-1-Topics-Questions.pdf:** Specific recent (2023-2024) question sets.
2.  **101-ielts-speaking-part-two-topics.pdf:** Comprehensive collection of Part 2 cue cards.
3.  **ielts-speaking-sample-tasks-2023.pdf:** Official sample tasks for test-taker preparation.

## RAG Prioritization Logic
- **Priority 1 [OFFICIAL]:** Use for core scoring, test structure transitions, and professional examiner conduct.
- **Priority 2 [PREPARATION]:** Use for supplemental question inspiration and topic variety.
